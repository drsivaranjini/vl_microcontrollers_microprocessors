## Theory

Mirrors the addition experiment: `22H` is loaded into the accumulator, `02H` into `R1`, and `SUBB A,R1`
subtracts `R1` (plus any incoming borrow — `R2` is pre-cleared so there isn't one) from the accumulator.
A borrow out of the subtraction clears the carry flag on the 8051's `SUBB`, so `JNC` here detects the
*borrow-occurred* case, and the difference plus a borrow flag are written to external memory the same
way as the addition experiment.

## Registers / Instructions Used

| Mnemonic         | Meaning                                                              |
| ----------------- | ------------------------------------------------------------------- |
| `MOV A,#22H`      | Load the minuend into the accumulator.                                |
| `MOV R1,#02H`     | Load the subtrahend into `R1`.                                        |
| `MOV R2,#00H`     | Initialise the borrow counter to 0.                                   |
| `SUBB A,R1`       | Subtract `R1` (and any incoming borrow) from the accumulator.         |
| `JNC Ahead`       | Skip the borrow increment if there was no borrow.                     |
| `INC R2`          | Record that a borrow occurred.                                        |
| `MOV DPTR,#9200H` | Point the data pointer at the result location.                        |
| `MOVX @DPTR,A`    | Write the accumulator (difference) to external memory.                |
| `INC DPTR`        | Advance the pointer to the next byte.                                  |
| `MOV A,R2`        | Move the borrow value into the accumulator to write it out.           |
| `MOVX @DPTR,A`    | Write the borrow to external memory.                                   |
| `LCALL 00BBH`     | Call the monitor's stop routine to halt execution.                     |

## Sample Program

<CodeBlock
  lang="asm"
  code={`MOV A, #22H       ; minuend
MOV R1, #02H      ; subtrahend
MOV R2, #00H      ; borrow = 0
SUBB A, R1        ; A = A - R1 (- any incoming borrow)
JNC Ahead         ; skip if no borrow
INC R2            ; record the borrow
Ahead:
MOV DPTR, #9200H  ; point at the result location
MOVX @DPTR, A     ; write the difference
INC DPTR
MOV A, R2         ; move borrow into A
MOVX @DPTR, A     ; write the borrow
LCALL 00BBH       ; stop`}
/>
