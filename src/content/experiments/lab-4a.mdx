## Theory

Two 8-bit operands (`23H` and `11H`) are loaded into the accumulator and `R1`. `ADD A,R1` sums them,
setting the carry flag if the result overflowed 8 bits. `JNC` skips the carry-increment when there was
no overflow. The sum is written to external memory at `9200H` via the data pointer, and the carry
(`0` or `1`) is written right after it at `9201H`.

## Registers / Instructions Used

| Mnemonic         | Meaning                                                              |
| ----------------- | ------------------------------------------------------------------- |
| `MOV A,#23H`      | Load the first operand into the accumulator.                         |
| `MOV R1,#11H`     | Load the second operand into `R1`.                                    |
| `MOV R2,#00H`     | Initialise the carry counter to 0.                                    |
| `ADD A,R1`        | Add `R1` into the accumulator; sets the carry flag on overflow.       |
| `JNC Ahead`       | Skip the carry increment if there was no overflow.                    |
| `INC R2`          | Record that a carry occurred.                                         |
| `MOV DPTR,#9200H` | Point the data pointer at the result location.                        |
| `MOVX @DPTR,A`    | Write the accumulator (sum) to external memory.                       |
| `INC DPTR`        | Advance the pointer to the next byte.                                  |
| `MOV A,R2`        | Move the carry value into the accumulator to write it out.            |
| `MOVX @DPTR,A`    | Write the carry to external memory.                                    |
| `LCALL 00BBH`     | Call the monitor's stop routine to halt execution.                     |

## Sample Program

<CodeBlock
  lang="asm"
  code={`MOV A, #23H       ; first operand
MOV R1, #11H      ; second operand
MOV R2, #00H      ; carry = 0
ADD A, R1         ; A = A + R1
JNC Ahead         ; skip if no carry
INC R2            ; record the carry
Ahead:
MOV DPTR, #9200H  ; point at the result location
MOVX @DPTR, A     ; write the sum
INC DPTR
MOV A, R2         ; move carry into A
MOVX @DPTR, A     ; write the carry
LCALL 00BBH       ; stop`}
/>

## Expected Output / Observation

| Address | Value | Meaning        |
| ------- | ----- | -------------- |
| `9200H` | `34H` | Sum (23H + 11H) |
| `9201H` | `00H` | Carry (none)    |
