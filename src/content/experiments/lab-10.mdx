## Theory

Keil µVision (and CPUlator, its free in-browser equivalent for this lab) assembles and runs standalone
ARM assembly. Each program below is a self-contained `AREA ... CODE, READONLY` block with an `ENTRY`
point and an infinite `B HERE` loop at the end (the standard way to halt a bare-metal ARM program so the
debugger can inspect final register state).

## Registers / Instructions Used

| Mnemonic       | Meaning                                                                |
| -------------- | --------------------------------------------------------------------- |
| `LDR R1,=0x...` | Load a full 32-bit literal into a register (via a literal pool).      |
| `MOV R1,#0x..`  | Move an 8-bit (rotated) immediate directly into a register.            |
| `ADD/SUB/MUL`   | Register-register arithmetic: `Rd, Rn, Rm`.                            |
| `CMP` / `BLT`   | Compare two registers; branch if the result was "less than".          |
| `B LOOP` / `B HERE` | Unconditional branch — used both for the division loop and to halt. |

## Sample Programs

### i. Addition

<CodeBlock
  lang="armasm"
  code={`AREA ARMex, CODE, READONLY   ; Name this block of code ARMex
ENTRY                        ; Mark first instruction to execute
main
  LDR R1, =0x00001133
  LDR R2, =0x00001111
  ADD R3, R1, R2
HERE
  B HERE                     ; End loop
  END                        ; End of file`}
/>

### ii. Subtraction

<CodeBlock
  lang="armasm"
  code={`AREA ARMex, CODE, READONLY
ENTRY
main
  MOV R1, #0x44
  MOV R2, #0x11
  SUB R3, R1, R2
HERE
  B HERE
  END`}
/>

### iii. Multiplication

<CodeBlock
  lang="armasm"
  code={`AREA ARMex, CODE, READONLY
ENTRY
main
  MOV R1, #0x5
  MOV R2, #0x2
  MUL R3, R1, R2
HERE
  B HERE
  END`}
/>

### iv. Division (quotient in R3, remainder in R0)

<CodeBlock
  lang="armasm"
  code={`AREA ARMex, CODE, READONLY
ENTRY
main
  MOV R0, #100          ; Dividend
  MOV R1, #10           ; Divisor
  MOV R3, #0            ; Quotient
LOOP
  CMP R0, R1
  BLT DONE
  ADD R3, R3, #1
  SUB R0, R0, R1
  B LOOP
DONE
HERE
  B HERE
  END`}
/>

## Expected Output / Observation

| Program        | Result register(s)                        |
| -------------- | ------------------------------------------- |
| Addition       | R3 = 0x00002244 (0x1133 + 0x1111)           |
| Subtraction    | R3 = 0x33 (0x44 − 0x11)                     |
| Multiplication | R3 = 0xA (5 × 2)                            |
| Division       | R3 = 10 (quotient), R0 = 0 (remainder, 100 ÷ 10) |
