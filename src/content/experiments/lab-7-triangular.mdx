## Theory

With `00h` at DAC2 giving 0V and `FFh` giving +5V, outputting data that slowly increases from `00` to
`FF` and then decreases back from `FF` to `00`, at regular intervals, produces a triangular wave: a
linear rise immediately followed by a linear fall.

| Port  | MICRO-51 | MICRO-POWER 1 | MICRO-31EB |
| ----- | -------- | ------------- | ---------- |
| DAC 1 | FFC0h    | A0C0h         | FFC0h      |
| DAC 2 | FFC8h    | A0C8h         | FFC8h      |

## Registers / Instructions Used

| Mnemonic        | Meaning                                                          |
| --------------- | ------------------------------------------------------------------ |
| `MOV DPTR,#FFC8` | Point the data pointer at DAC2.                                    |
| `MOVX @DPTR,A`   | Write the accumulator to DAC2.                                     |
| `INC A` / `DEC A` | Step the ramp up (INC) or down (DEC).                             |
| `JNZ LOOP1`      | Loop while `A ≠ 0` — keeps ramping up until it wraps `FFh → 00h`.  |
| `LJMP START`     | Long jump back to the very start once a full up/down cycle completes. |

## Sample Program

### To obtain 0V at DAC 1 (reference)

<CodeBlock
  lang="asm"
  code={`ORG 4100
MOV A,#3F
MOV DPTR,#FFC0
MOVX @DPTR,A
HERE: SJMP HERE`}
/>

### Triangular waveform at DAC 2

<CodeBlock
  lang="asm"
  code={`ORG 4100
MOV DPTR,#FFC8
START:
MOV A,#00
LOOP1: MOVX @DPTR,A   ; ramp up
INC A
JNZ LOOP1             ; loop until A wraps to 0
MOV A,#FF
LOOP2: MOVX @DPTR,A   ; ramp down
DEC A
JNZ LOOP2             ; loop until A reaches 0
LJMP START`}
/>

## Expected Output / Observation

DAC2's output rises linearly from 0V to +5V then falls linearly back to 0V, repeating continuously — a
symmetric triangular wave, with a brief single-sample plateau at each peak/trough (the last write of one
ramp and the first write of the next are both the extreme value, `FF` or the initial `00`).
