## Theory

With `00h` at DAC2 giving ŌłÆ5V and `FFh` giving +5V, writing `00` and `FF` alternately at DAC2, with a
fixed delay between writes, produces a square wave whose period is set by the delay routine.

| Port  | MICRO-51 | MICRO-POWER 1 | MICRO-31EB |
| ----- | -------- | ------------- | ---------- |
| DAC 1 | FFC0h    | A0C0h         | FFC0h      |
| DAC 2 | FFC8h    | A0C8h         | FFC8h      |

## Registers / Instructions Used

| Mnemonic       | Meaning                                                                 |
| -------------- | -------------------------------------------------------------------------- |
| `LCALL DELAY`  | Long call to a delay subroutine (any 16-bit address).                       |
| `MOV R1,#05`   | Outer delay counter.                                                       |
| `DJNZ R2,HERE` | Decrement `R2`; loop to `HERE` while nonzero ŌĆö the inner delay loop.       |
| `DJNZ R1,LOOP` | Decrement `R1`; repeats the inner loop 5 times for a longer total delay.  |
| `RET`          | Return from the delay subroutine to the caller.                           |

## Sample Program

### To obtain 0V at DAC 1 (reference)

<CodeBlock
  lang="asm"
  code={`ORG 4100
MOV A,#3F
MOV DPTR,#FFC0
MOVX @DPTR,A
HERE: SJMP HERE`}
/>

### Square waveform at DAC 2

<CodeBlock
  lang="asm"
  code={`ORG 4100
MOV DPTR,#FFC8
START: MOV A,#00
MOVX @DPTR,A
LCALL DELAY
MOV A,#FF
MOVX @DPTR,A
LCALL DELAY
LJMP START

DELAY: MOV R1,#05
LOOP:  MOV R2,#FF
HERE:  DJNZ R2,HERE
       DJNZ R1,LOOP
       RET`}
/>

## Expected Output / Observation

DAC2 alternates between ŌłÆ5V and +5V at a period set by the `DELAY` subroutine (5 ├Ś 255 inner
iterations per half-cycle) ŌĆö a bipolar square wave.
